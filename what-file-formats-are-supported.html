What file formats are supported?

Hi and thank you for choosing Twayobiz. We support the following file formats.

Image formats

  • PNG
  • JPG/JPEG
  • GIF
  • WEBP
  • SVG

Max size: 25MB

Document formats

  • TXT
  • PDF
  • Microsoft Office files: DOC, DOCX, DOCB, DOT, DOTX, XLS, XLSX, XLT, XLTX, PPT, PPTX, POT, POTX, PPS, PPSX
  • CSV

Max size: 1000MB

Audio formats

  • MP3
  • WMA

Max size: 50MB

Video formats

  • MP4
  • MOV
  • MKV

Max size: 4GB

Vector formats

  • SVG

Max size: 250KB
